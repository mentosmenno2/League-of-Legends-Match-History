# League of Legends Match History
### Eindopdracht Serious JS

League of Legends is een spel, met een eigen API voor developers. Deze is sinds een paar maanden opengesteld voor iedereen. Hiermee is bijna alle informatie van spelers, karakters, geschiedenis van games en meer op te halen. Van deze informatie wil ik een aantal componenten gebruiken om een match history list te maken.

Als eerste kan je zoeken of een user bestaat. Hierbij zal ik dit testen met mijn eigen account, waarvan de summoner name “mentozer” is. Vervolgens is het de bedoeling om een lijst te laten zien met alle matches die ik heb gespeeld in de afgelopen tijd.

Uit deze lijst is het mogelijk om een match te kiezen, om daarvan de details te bekijken. Hierbij worden er verschillende onderdelen van de match getoond. Het belangrijkste onderdeel hiervan zijn de scoreresultaten per speler (Kill/Dead/Assist-ratio). Hieraan is precies op te maken hoe goed elke speler gespeeld heeft in die match. Daarnaast is er per spelers te zien welke items ze hebben gekocht in die match, en welke 2 speciale krachten (summoner spells) ze hebben gebruikt in die match.

Daarnaast is het mogelijk om de totale eindstand van de game te zien, met de algemene resultaten.

In de lijst met resultaten per speler is het ook mogelijk om naar een profiel van een speler te gaan, door op deze speler te klikken. Hier kan je dan zijn ranks zien, meest gespeelde karakters en veel meer.
